How do access labs?

We have built many labs and cover most of the vulnerabilities. Click on the Labs tab from the top of the screen. Select the type of vulnerability which you would like to solve.

I am not able to solve lab. What should I do?

Labs are designed for begineers. Please try a bit harder. We have added hints, try to solve labs refering those. If you still are not able to figure out, we also have solutions to the labs. In case you arent able to understand solution, you can post your doubt on the discussion forum.

What is OWASP?

The Open Web Application Security Project (OWASP) is a non-profit organization founded in 2001, with the goal of helping website owners and security experts protect web applications from cyber attacks. OWASP has 32,000 volunteers around the world who perform security assessments and research.
